Privacy Policy

Discogs Theme Designer does not collect, store, transmit, or share any personal data.

The extension operates entirely locally in the user's browser.

The only data stored is user-selected theme preferences, which are saved locally using Chrome storage to ensure settings persist across page reloads.

No analytics, tracking, advertising, or external network requests are used.

This extension modifies only the visual appearance of the Discogs website.

If you have any questions, you can contact the developer via the Chrome Web Store listing.
